# Fuzzy Nearness Spaces for Enhanced Edge Detection and Tumor Classification in Medical Imaging

## Overview
This repository contains the complete MATLAB implementation for reproducing all results 
reported in the paper "Fuzzy Nearness Spaces for Enhanced Edge Detection and Tumor 
Classification in Medical Imaging."

## Requirements
- **MATLAB 2024b** (or later)
- **Toolboxes Required:**
  - Image Processing Toolbox
  - Statistics and Machine Learning Toolbox
- **Hardware Used:** Intel Core i7-10700K @ 3.8GHz, 32GB DDR4, NVIDIA RTX 3080

## Datasets
Download the following datasets and place them in the appropriate directories:

1. **Chest X-ray Pneumonia Dataset**
   - Source: https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia
   - 5,856 images (JPEG format)

2. **MIAS Mammographic Dataset**
   - Source: https://www.kaggle.com/datasets/kmader/mias-mammography
   - 322 images (PGM format, 1024×1024 pixels)
